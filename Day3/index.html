Understanding CSS Position

The position property controls how an element is placed on the page. Static is the default — elements just stack in normal document flow. Relative lets you nudge an element while its original space stays reserved. Absolute removes it from the flow completely, so it can overlap other elements. Fixed locks it to the viewport, so it stays in place even while the page scrolls.

A common place you'll see relative and absolute working together is an author card, where a small status badge is pinned to the corner of a profile picture. You'll also often see a fixed "Back to Top" button in the corner of the screen on long blog pages like this one.
